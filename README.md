# Pymes\_TP

Proyecto realizado en clase de Desarrollo de Software (2025).

-- Tecnologías utilizadas:
--> Backend: NodeJs, Express , WebApiRest, Swagger, Sequelize, Sqlite y Javascript.

--> Frontend: Single Page Aplication, HTML, CSS, Bootstrap, NodeJs, Javascript y React.

Para inicializar proyecto es necesario ejecutar el comando "npm install" en la carpeta del Backend y Frontend, para posteriormente ejecutar "npm run dev" y poder visualizar en un host local la pagina completa.
